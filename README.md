# FSAD Skill Workbook

This repository contains a collection of **Full Stack Application Development (FSAD)** skill experiments, covering backend, frontend, and full-stack concepts using modern technologies.

---

## Skill Experiments Overview

### Skill 1: Git Version Control

Implemented version control using Git, including repository setup, commits, branching, merging, and conflict resolution.

### Skill 2: Hibernate CRUD Operations

Developed a product management system using Hibernate and JPA annotations to perform complete CRUD operations.

### Skill 3: HQL Operations

Performed advanced data retrieval using Hibernate Query Language (HQL), including sorting, pagination, filtering, and aggregate functions.

### Skill 4: Spring Dependency Injection

Demonstrated Constructor and Setter Injection using both XML and annotation-based configurations in Spring.

### Skill 5: Spring Autowiring

Implemented dependency injection using `@Autowired` for automatic bean wiring within the Spring IoC container.

### Skill 6: Spring MVC Request Handling

Built RESTful endpoints using Spring MVC to handle HTTP requests, path variables, and request parameters.

### Skill 7: REST API with ResponseEntity

Designed REST APIs with proper HTTP status handling using `ResponseEntity` for structured responses.

### Skill 8: JPQL & Query Methods

Implemented dynamic data retrieval using Spring Data JPA with derived query methods and JPQL.

### Skill 9: Global Exception Handling

Handled application-wide exceptions using `@ControllerAdvice` and custom exception classes to improve API responses.

### Skill 10: React State Management

Developed a dynamic React component using `useState` to manage and update UI state efficiently.

### Skill 11: React API Integration

Integrated APIs using Fetch API and Axios, including local JSON, public APIs, and component-based navigation.

### Skill 12: Full Stack CRUD Application

Built a full-stack application integrating React frontend with Spring Boot backend for seamless CRUD operations.

### Skill 13: Deployment

Deployed a full-stack application by generating production builds and hosting backend services.

### Skill 14: Authentication & Session Management

Implemented user authentication with login, registration, session handling, protected routes, and logout functionality.

### Skill 15: JWT Authentication & Authorization

Secured APIs using JWT tokens and implemented role-based access control with Spring Security.

### Skill 16: Swagger API Documentation

Integrated Swagger/OpenAPI to document, visualize, and test REST APIs with detailed request and response structures.
